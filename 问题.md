# 过程中遇到的问题

### 1、阶段1的disc趋于0速度过快
![问题1](./images/问题1.png)

##### 解决方法：

```bash
问题明确了：判别器第5 epoch 就死透了，生成器拿不到对抗梯度，VAE 只学了模糊平均，后面的 LDT 自然全废。

我来做三件事修复它：

判别器加 R1 梯度惩罚 — 防止判别器过强
标签平滑 — 不让判别器完美收敛
分离 G/D 学习率 — D 用更低的学习率
```
